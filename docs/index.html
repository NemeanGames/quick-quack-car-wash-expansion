Quick Quack Car Wash Expansion Strategy

This project analyses the expansion of Quick Quack Car Wash by exploring competitor presence, growth trends, revenue models, customer sentiment, and geographic coverage. The analysis is reproducible using R Markdown and visualised via a React front‑end.

Data Sets

The repository includes sample CSV datasets used in the analysis:

  • competitor_dataset.csv – competitor count by state
  • competitor_growth_trend.csv – growth trends of competitor locations
  • membership_roi.csv – membership revenue and ROI projections
  • mod_wash_forecast.csv – Mod Wash forecast and expansion metrics
  • sentiment_reviews.csv – summary of customer sentiment from reviews

Data files reside in the data folder of the repository.

Interactive Map

The front‑end component (React + Leaflet) visualises car wash locations on an interactive map. To run the app locally, see instructions in the README.

Report

An R Markdown report (report.Rmd) demonstrates how to knit the analysis into a static HTML report. To build the report locally, run:

Rscript scripts/render_report.R

It will create a docs/ folder and write index.html inside for deployment on GitHub Pages.

This static page serves as a placeholder on GitHub Pages. Clone the repository and follow the README to build the full report and application.
